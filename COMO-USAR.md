# LC Viagens — Como Usar o Site

## 🌐 ABRIR O SITE LOCALMENTE

1. Abra a pasta `lc-viagens`
2. Dê duplo clique em **`index.html`**
3. O site abre no navegador!

---

## ✏️ EDITAR O SITE (MODO ADMIN)

### Passo 1 — Abrir o painel admin
- Acesse: `admin/login.html`
- **E-mail:** admin@lcviagens.com.br
- **Senha:** LCViagens@2026

### Passo 2 — Editar elementos
- Clique em qualquer texto, imagem ou botão para editar
- Uma janela de edição aparece no canto direito
- Faça as alterações e clique em **✓ Aplicar**

### Passo 3 — Publicar
- Clique no botão verde **🚀 Publicar** na barra do topo
- As alterações ficam visíveis para todos em ~30 segundos

---

## ➕ ADICIONAR NOVO PACOTE

1. Entre no modo admin
2. Clique em **➕ Pacote** na barra do topo
3. Preencha todos os dados do pacote
4. Clique em **✓ Criar Pacote**
5. Publique para salvar permanentemente

---

## 🖼️ TROCAR A LOGO

1. Substitua o arquivo `imagens/logo_lc.jfif` pela logo real
2. Ou entre no modo admin e clique na logo para trocar

## 🚀 PUBLICAR NA INTERNET (VERCEL)

1. Crie uma conta em vercel.com
2. Crie um repositório no GitHub com estes arquivos
3. Conecte o repositório ao Vercel
4. Configure as variáveis de ambiente (ver REGISTRO-PROJETO.md)
5. Pronto! Site no ar gratuitamente
